# 三角喷涂 - 山形区间叠加问题题解

## 题目简述
长度为 `n` 的数组初始全为 0，共 `m` 次操作。每次给出区间 `[l, r]`，对区间内位置 `i` 增加 `min(i - l + 1, r - i + 1)`，即叠加一个对称山形增量：端点处为 1，向中间逐次 +1 到峰值后对称递减。求所有操作完成后数组的最终值。

### 输入输出
- 输入：多组测试数据。每组先输入 `n, m`，随后 `m` 行每行两个整数 `l, r`。
- 输出：每组输出一行 `n` 个整数，对应数组每个位置的最终数值。
- 数据范围：所有测试的 `n` 之和、`m` 之和均不超过 $2\times10^5$。

## 解题思路
### 一阶差分的局限
直观做法是用一阶差分处理区间加法，但本题增量是先增后减的等差数列，若逐区间更新一阶差分，时间复杂度退化为 $O(m\times n)$，会直接超时。

### 核心优化：二阶差分
山形增量的一阶差分呈分段常数变化，因此可以用**二阶差分数组**，将每次区间操作转化为常数次点修改，最终通过两次前缀和还原原数组，整体时间复杂度优化为 $O(n + m)$。

#### 原理推导
对于山形区间 `[l, r]`：
- 原数组：从 `l` 开始逐次 +1，到中点达峰值，之后逐次 -1，到 `r` 处为 1
- 一阶差分：左半段为 +1，中点后变为 -1，区间外为 0
- 二阶差分：仅在几个关键位置跳变，其余位置均为 0

令 `len = r - l`，`mid = (l + r) / 2`，分两种情况修改二阶差分：

##### 情况1：区间长度为偶数（len 为奇数）
例：区间 `[2, 5]`
| 位置 i | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|---|---|---|---|---|---|---|
| 原数组 | 0 | 1 | 2 | 2 | 1 | 0 | 0 |
| 一阶差分 | 0 | 1 | 1 | 0 | -1 | -1 | 0 |
| 二阶差分 | 0 | 1 | 0 | -1 | -1 | 0 | 1 |

修改规则：
- `diff2[l] += 1`
- `diff2[mid+1] -= 1`
- `diff2[mid+2] -= 1`
- `diff2[r+2] += 1`

##### 情况2：区间长度为奇数（len 为偶数）
例：区间 `[2, 4]`
| 位置 i | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|---|---|---|---|---|---|---|
| 原数组 | 0 | 1 | 2 | 1 | 0 | 0 | 0 |
| 一阶差分 | 0 | 1 | 1 | -1 | -1 | 0 | 0 |
| 二阶差分 | 0 | 1 | 0 | -2 | 0 | 1 | 0 |

修改规则：
- `diff2[l] += 1`
- `diff2[mid+1] -= 2`
- `diff2[r+2] += 1`

### 算法步骤
1. 初始化二阶差分数组 `diff2` 为 0
2. 对每次操作，按区间长度奇偶性修改二阶差分对应位置
3. 第一次前缀和得到一阶差分数组
4. 第二次前缀和得到最终原数组

## 完整代码
```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

void solve() {
    int n, m;
    cin >> n >> m;
    vector<int> diff2(n + 3, 0); // 开 n+3 防止 r+2 越界
    
    for(int i = 0; i < m; i++) {
        int l, r;
        cin >> l >> r;
        int len = r - l;
        int mid = (l + r) / 2;
        
        if(len & 1) {
            diff2[l]++;
            diff2[mid + 1]--;
            diff2[mid + 2]--;
            diff2[r + 2]++;
        } else {
            diff2[l]++;
            diff2[mid + 1] -= 2;
            diff2[r + 2]++;
        }
    }

    int diff1 = 0, ans = 0;
    for(int i = 1; i <= n; i++) {
        diff1 += diff2[i]; // 一阶前缀和
        ans += diff1;      // 二阶前缀和
        if(i != 1) cout << ' ';
        cout << ans;
    }
    cout << '\n';
}

signed main() {
    cin.tie(0)->sync_with_stdio(0);
    int T;
    cin >> T;
    while(T--) solve();
    return 0;
}
```

## 避坑指南
1. **数组越界**：修改点涉及 `r+2`，差分数组至少开到 `n+2`，建议 `n+3` 留余量。
2. **奇偶判断**：判断条件是 `r - l` 的奇偶性，与区间长度奇偶性差 1，切勿搞反。
3. **中点位置**：中点用整数除法 `(l+r)/2`，建议先在草稿纸举小例子验证后再编码。

## 解题心得
- 区间叠加类问题可按增量阶数选择差分方案：常数增量用一阶差分，线性增量用二阶差分，可大幅降低复杂度。
- 差分的加减位置容易混淆，举小例子列清一、二阶差分表格，验证正确后再编码更稳妥。
- 多在草稿纸上尝试，搞清楚细节变化。
