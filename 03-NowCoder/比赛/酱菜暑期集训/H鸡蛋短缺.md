# H. 鸡蛋短缺
> 做题时间：2026-07-20

> 竞赛：酱菜暑期集训（2025年ICPC新疆维吾尔自治区大学生程序设计竞赛重现赛）

> 原题链接：https://ac.nowcoder.com/acm/contest/138195/H

## 题目简述
一共有 `n` 种不同品类的鸡蛋，超市提供 `m` 份鸡蛋套装，每套套装包含若干种鸡蛋。我们可以任选若干套套装购买，目标是集齐全部 `n` 种鸡蛋。
请求出**最少需要购买多少套**才能凑齐所有鸡蛋；若无论怎么选都无法集齐全部品类，输出 `-1`。

## 解题核心思路：二进制状态压缩 + 01背包求最小选取数
1. **二进制状态压缩表示鸡蛋库存**
    用二进制数字 `state` 代表当前已拥有的鸡蛋种类：二进制第 `x` 位为 `1` 代表已经拥有第 `x` 种鸡蛋，为 `0` 代表缺少该品类。
    - 全集状态 `up = (1 << n) - 1`：二进制后 `n` 位全为1，代表集齐全部鸡蛋。
    - 每套套装也预处理成一个二进制数 `a[i]`，标记该套装包含哪些鸡蛋。
2. **01背包模型转化**
    经典01背包一般求最大价值/容量，本题变形为：从 `m` 个物品（套装）中选若干件，物品作用是**按位或扩充已有鸡蛋集合**，求到达全集状态的**最少选取件数**。
    - 定义 `dp[state]`：凑出鸡蛋状态 `state` 所需购买的最少套装数量；
    - 初始 `dp[0] = 0`（一套不买，无任何鸡蛋），其余状态初始化为无穷大 `m+1`；
    - 背包逆序遍历所有状态，对每个套装执行状态转移：`new_state = state | a[i]`，用 `dp[state] + 1` 更新 `dp[new_state]` 的最小值。
3. **无解判定**
    把所有套装做按位或得到总集合 `s`，若 `s < up`，说明所有套装合起来都凑不齐全部鸡蛋，直接输出 `-1`。

## 完整代码
```cpp
#include<bits/stdc++.h>
#define all(A) A.begin(),A.end()
using namespace std;
int main() {
    int n, m;
    cin >> n >> m;
    // 二进制存储完整鸡蛋全集状态
    int up = (1<<n)-1;
    vector<int> a(m, 0);
    int s = 0;
    for(int i=0; i<m; i++) {
        int z; cin >> z;
        for(int j=0; j<z; j++) {
            int x; cin >> x;
            a[i] += (1<<(x-1));
        }
        s |= a[i];
    }
    // 所有套装合并仍无法集齐全部鸡蛋，无解
    if(s < up) {
        cout << -1 << '\n';
        return 0;
    }
    // 状态压缩01背包，dp[state]代表凑出该状态最少套数
    vector<int> dp(up+1, m+1);
    dp[0] = 0;
    for(int i=0; i<m; i++) {
        for(int state=up; state>=0; state--) {
            if(dp[state] != m+1) {
                int new_state = state | a[i];
                dp[new_state] = min(dp[new_state], dp[state] + 1);
            }
        }
    }
    cout << dp[up] << '\n';
}
```
