# 二进制的有趣玩法
前言：认识理解二进制后会发现二进制有很多好玩的地方，也能体会到当进制不是二时的实际含义，还可以解锁关于进制的玩法

## 二进制的01
用`1`/`0`表示 有/没有；选/不选。
好处是能将物品拥有状态转化为数字，对比状态就是对比数字，还可以用来做小范围的排列组合；弊端是数据量不能太大。

运用举例：[H鸡蛋短缺](../03-NowCoder/比赛/酱菜暑期集训/H鸡蛋短缺.md)

### 排列组合中的运用（位掩码法）
```cpp
// 枚举所有大小为 k 的子集
void combination(const vector<int>& nums, int k) {
    int n = nums.size();
    for (int mask = 0; mask < (1 << n); ++mask) {
        if (__builtin_popcount(mask) != k) continue;
        // 输出当前组合
        for (int i = 0; i < n; ++i) {
            if (mask >> i & 1) {
                cout << nums[i] << " ";
            }
        }
        cout << "\n";
    }
}
```
