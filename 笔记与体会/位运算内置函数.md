# GCC 位运算内置函数速查笔记
> 适用于算法竞赛 / C++ 位运算优化，为 GCC/Clang 编译器内置扩展函数，由硬件指令实现，时间复杂度 O(1)，远快于手写循环。

---

## 版本命名规则
所有函数均提供 32 位与 64 位两个版本：
- 无后缀：处理 32 位无符号整数（`unsigned int`）
- 带 `ll` 后缀：处理 64 位无符号整数（`unsigned long long`）

> 处理 `long long` 类型数据时，必须使用 `ll` 后缀版本，否则会出现结果错误。

---

## 1. 前导零计数：`__builtin_clz` / `__builtin_clzll`
### 全称
Count Leading Zeros (long long)

### 功能
统计整数二进制表示中，**最高位 1 左侧连续的 0 的个数**（前导零数量）。

### 函数原型
```cpp
int __builtin_clz(unsigned int x);
int __builtin_clzll(unsigned long long x);
```

### 示例
```cpp
// x = 5，二进制：...000101
__builtin_clz(5);    // 32 位下返回 29（前29个0，后3位有效）
__builtin_clzll(5);  // 64 位下返回 61（前61个0，后3位有效）
```

### 典型用法
计算整数的二进制有效位数：
```cpp
// 32 位
int bit_len = 32 - __builtin_clz(x);
// 64 位
int bit_len_ll = 64 - __builtin_clzll(x);
```

### 注意事项
- 当 `x = 0` 时，结果为**未定义行为**，不同平台返回值不一致，使用前必须保证 `x > 0`。

## 2. 1 的个数计数：`__builtin_popcount` / `__builtin_popcountll`
### 全称
Population Count (long long)

### 功能
统计整数二进制表示中，**数值为 1 的位的总个数**（又称汉明重量 / Population Count）。

### 函数原型
```cpp
int __builtin_popcount(unsigned int x);
int __builtin_popcountll(unsigned long long x);
```

### 示例
```cpp
// x = 13，二进制：1101
__builtin_popcount(13);    // 返回 3（共3个1）
__builtin_popcountll(13);  // 返回 3
```

### 典型场景
- 判断一个数是否为 2 的幂：`x > 0 && __builtin_popcount(x) == 1`
- 状态压缩 DP 中统计二进制状态的 1 的数量
- 位运算枚举、子集计数

### 注意事项
- `x = 0` 时合法，返回 0。

---

## 3. 尾随零计数：`__builtin_ctz` / `__builtin_ctzll`
### 全称
Count Trailing Zeros (long long)

### 功能
统计整数二进制表示中，**最低位 1 右侧连续的 0 的个数**（尾随零数量）。

### 函数原型
```cpp
int __builtin_ctz(unsigned int x);
int __builtin_ctzll(unsigned long long x);
```

### 示例
```cpp
// x = 12，二进制：...1100
__builtin_ctz(12);    // 返回 2（末尾有2个0）
__builtin_ctzll(12);  // 返回 2
```

### 典型场景
- 快速求 `lowbit` 对应的位置
- 分解整数中 2 的因子个数
- 树状数组、位运算优化辅助计算

### 注意事项
- 当 `x = 0` 时，结果为**未定义行为**，使用前必须保证 `x > 0`。

---

## 4. 最低位 1 定位：`__builtin_ffs` / `__builtin_ffsll`
### 全称
Find First Set (long long)

### 功能
返回整数二进制中**最低位 1 的位置**（从 1 开始计数）；若 `x = 0`，则返回 0。

### 函数原型
```cpp
int __builtin_ffs(int x);
int __builtin_ffsll(long long x);
```

### 示例
```cpp
// x = 12，二进制：...1100
__builtin_ffs(12);    // 返回 3（最低位1在第3位）
__builtin_ffsll(12);  // 返回 3
__builtin_ffs(0);     // 返回 0
```

### 与 `__builtin_ctz` 的关系
当 `x ≠ 0` 时：
```
__builtin_ffs(x) = __builtin_ctz(x) + 1
```

### 注意事项
- 参数为有符号整数，`x = 0` 时行为定义明确，返回 0，安全性优于 `__builtin_ctz`。

---

## 统一注意事项
1. **0 值未定义行为**：`__builtin_clz`、`__builtin_clzll`、`__builtin_ctz`、`__builtin_ctzll` 在参数为 0 时结果不可控，使用前务必做非 0 判断。
2. **参数类型匹配**：传入有符号负数时，会先转换为无符号再计算，结果可能不符合预期，建议显式强转为无符号类型。
3. **兼容性**：属于 GCC/Clang 编译器扩展，不属于 C++ 标准，MSVC 环境无法直接使用；算法竞赛环境（洛谷、Codeforces、NOIP 等）均支持。
4. **常数优化**：所有函数均为硬件指令级实现，无函数调用开销，是位运算场景的标准优化手段。

## 常用技巧速查表
| 需求 | 代码写法（32 位） |
|------|-------------------|
| 求 x 的二进制有效位数 | `32 - __builtin_clz(x)` |
| 判断 x 是否为 2 的幂 | `x > 0 && __builtin_popcount(x) == 1` |
| 求最低位 1 的位置（从 0 开始） | `__builtin_ctz(x)` |
| 求最低位 1 的位置（从 1 开始） | `__builtin_ffs(x)` |
| 统计二进制中 1 的个数 | `__builtin_popcount(x)` |
